Account Book

[website]

An online account book application based on React, React-redux and Firebase

  • Designed a single page web application with React and React-redux to access and operate the user expenses.

  • Developed an interactive web page utilizing AJAX technology (HTML, CSS and JavaScript) to add, edit, remove, sort and filter the expenses data.

  • Implemented client-side routing using React-Router to navigate among views of various components.

  • Used Google Firebase to support authentication, upload, delete and search functionality.

  • Deployed the account book application to Heroku for demonstration.
